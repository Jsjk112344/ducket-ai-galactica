1. Install dependency:
   ```
   cd agent && npm install @anthropic-ai/sdk
   ```

2. Create `agent/src/classify.js` with these exports:
   - `classifyListing(listing)` — async, returns `{ category, confidence, reasoning, classificationSource }`
   - `classifyByRules(listing)` — exported for testing

3. Rule-based first pass (`classifyByRules`):
   - If `priceDeltaPct > 100`: category = `SCALPING_VIOLATION`, confidence = `Math.min(95, 70 + Math.round(priceDeltaPct / 20))`, reasoning references the exact priceDeltaPct value
   - If `priceDeltaPct < -10`: category = `LIKELY_SCAM`, confidence = 72, reasoning references below-face-value pattern
   - If `redFlags` array contains entries with 'new seller' or 'no proof': category = `COUNTERFEIT_RISK`, confidence = 68
   - Otherwise: category = `LEGITIMATE`, confidence = 55, reasoning = 'No strong rule signals detected.'
   - All rule results include `classificationSource: 'rules'`

4. Claude API classification (`classifyWithClaude`):
   - Use `import Anthropic from '@anthropic-ai/sdk'`
   - Model: `process.env.CLAUDE_MODEL ?? 'claude-sonnet-4-6'`
   - System prompt: fraud detection rubric listing all 4 categories with signal descriptions
   - Use `output_config.format` with `json_schema` type, schema enforcing:
     - `category`: string enum of the 4 categories
     - `confidence`: number 0-100
     - `reasoning`: string
     - `required: ['category', 'confidence', 'reasoning']`, `additionalProperties: false`
   - Parse response: `response.content.find(b => b.type === 'text')?.text`
   - Add `classificationSource: 'claude'` to result

5. Main `classifyListing(listing)` logic:
   - Run `classifyByRules(listing)` first
   - If `listing.source === 'mock'`: return rules result with `classificationSource: 'rules-mock'`, skip Claude
   - If rules confidence >= 85: return rules result as-is, skip Claude
   - Otherwise: try Claude API, catch errors and fallback to rules with `classificationSource: 'rules-only'`
   - Log to stderr: `[Classify] {platform} | {category} | {confidence}% | source:{classificationSource}`

6. Use ESM format with `import`/`export`. Load dotenv at module top matching existing pattern:
   ```javascript
   import * as dotenv from 'dotenv';
   import { join, dirname } from 'node:path';
   import { fileURLToPath } from 'url';
   const __dirname = dirname(fileURLToPath(import.meta.url));
   dotenv.config({ path: join(__dirname, '../../.env'), quiet: true });
   ```

7. Add `CLAUDE_API_KEY` and `CLAUDE_MODEL=claude-sonnet-4-6` and `FRAUD_CONFIDENCE_THRESHOLD=85` to `.env.example` if not already present (append, do not overwrite existing lines).
