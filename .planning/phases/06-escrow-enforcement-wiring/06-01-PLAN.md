Create `agent/src/escrow.js` as an ESM module with these specifics:

**Imports and constants:**
- `import { ethers } from 'ethers'` — for Interface, keccak256, toUtf8Bytes, Wallet, JsonRpcProvider, Contract
- `import { createRequire } from 'node:module'` — for loading JSON artifacts (createRequire pattern for ESM)
- `import { getAccount } from './wallet/index.js'` — WDK account
- Load ABI: `const require = createRequire(import.meta.url)` then `const artifact = require('../../contracts/artifacts/src/FraudEscrow.sol/FraudEscrow.json')`
- Load deployed addresses: `const deployed = require('../../contracts/deployed.json')`
- `const ESCROW_INTERFACE = new ethers.Interface(artifact.abi)`
- `const ESCROW_ADDRESS = deployed.sepolia.FraudEscrow` (value: `0x6427d51c4167373bF59712715B1930e80EcA8102`)
- `const USDT_ADDRESS = deployed.sepolia.usdt` (value: `0x7169d38820dfd117c3fa1f22a697dba58d90ba06`)
- `const USDT_DECIMALS = 6`
- `const ESCROW_AMOUNT = 10n * (10n ** BigInt(USDT_DECIMALS))` — fixed 10 USDT per user decision
- `const BOUNTY_POOL = process.env.BOUNTY_POOL_ADDRESS ?? ESCROW_ADDRESS` — fallback to contract itself as pool
- `const log = (...args) => process.stderr.write('[Escrow] ' + args.join(' ') + '\n')` — stderr per project pattern

**Helper: waitForConfirmation(account, hash, maxAttempts=10, delayMs=2000)**
- Poll `account.getTransactionReceipt(hash)` up to maxAttempts
- Return receipt on success, throw Error on timeout

**Helper: getOwnerSigner()**
- `new ethers.JsonRpcProvider(process.env.SEPOLIA_RPC_URL)`
- `new ethers.Wallet(process.env.SEPOLIA_DEPLOYER_PRIVATE_KEY, provider)`
- This is for onlyOwner calls (release/refund/slash) since FraudEscrow.owner() is the deployer address

**Helper: makeEscrowId(listingUrl, timestamp)**
- `ethers.keccak256(ethers.toUtf8Bytes('listing:' + listingUrl + ':' + timestamp))`
- Returns bytes32

**Helper: makeBondEscrowId(eventName)**
- `ethers.keccak256(ethers.toUtf8Bytes('bond:' + eventName))`

**Export: depositEscrow({ escrowId, isBond = false })**
1. Get WDK account via `getAccount(0)`
2. Check balance: `account.getTokenBalance(USDT_ADDRESS)` — if < ESCROW_AMOUNT, log warning, return null
3. Approve: `account.approve({ token: USDT_ADDRESS, spender: ESCROW_ADDRESS, amount: ESCROW_AMOUNT })`
4. Wait for approve confirmation via waitForConfirmation
5. Encode deposit calldata: `ESCROW_INTERFACE.encodeFunctionData('deposit', [escrowId, ESCROW_AMOUNT])`
6. Submit via WDK: `account.sendTransaction({ to: ESCROW_ADDRESS, value: 0n, data })`
7. Log with `[Escrow]` prefix: txHash, Etherscan link, amount, "deposit" action type
8. Return `{ txHash: result.hash, escrowId, etherscanLink }`
9. Wrap in try/catch — on error: log error, if caseFilePath known call `updateCaseFileEscrow(caseFilePath, { escrowStatus: 'failed', error: err.message })` to mark failure in case file, then return null (demo never crashes per user decision)

**Export: releaseEscrow({ escrowId, recipientAddress })**
1. Get owner signer via getOwnerSigner()
2. Create contract instance: `new ethers.Contract(ESCROW_ADDRESS, artifact.abi, signer)`
3. Call `contract.release(escrowId, recipientAddress)`
4. `await tx.wait(1)` for confirmation
5. Log with `[Escrow]` prefix
6. Return `{ txHash, etherscanLink }` or null on error

**Export: refundEscrow({ escrowId })**
- Same pattern as release but calls `contract.refund(escrowId)`

**Export: slashEscrow({ escrowId, bountyPool })**
- Same pattern as release but calls `contract.slash(escrowId, bountyPool ?? BOUNTY_POOL)`

**Export: dispatchEscrowAction({ category, listing, caseFilePath, timestamp })**
1. Generate escrowId via makeEscrowId(listing.url, timestamp)
2. First call depositEscrow to lock funds
3. If deposit fails (returns null), return null
4. Then dispatch based on category:
   - `SCALPING_VIOLATION` -> slashEscrow({ escrowId, bountyPool: BOUNTY_POOL })
   - `LIKELY_SCAM` -> refundEscrow({ escrowId })
   - `COUNTERFEIT_RISK` -> refundEscrow({ escrowId })
   - `LEGITIMATE` -> releaseEscrow({ escrowId, recipientAddress: listing.sellerAddress ?? BOUNTY_POOL })
5. If action succeeds, call updateCaseFileEscrow (imported from evidence.js)
6. Log per-cycle summary line
7. Return result with txHash and etherscanLink

**Add to agent/src/evidence.js — new export `updateCaseFileEscrow(filepath, txHash, action)`:**
- Import `readFile` (add to existing import from 'node:fs/promises')
- Read the case file content
- Replace `| Etherscan Link | _(pending escrow transaction)_ |` with `| Etherscan Link | [${txHash.slice(0,10)}...](https://sepolia.etherscan.io/tx/${txHash}) |`
- Replace `| Action Taken | ${action} |` with `| Action Taken | ${action} (confirmed) |`
- Write back with writeFile
- Log to stderr: `[Evidence] Case file updated with Etherscan link: ${txHash.slice(0,10)}...`

**Anti-patterns to avoid (per RESEARCH.md):**
- Do NOT use `account._account` to access the HDNodeWallet directly — always use `account.sendTransaction()`
- Do NOT create ethers.Wallet from ESCROW_WALLET_SEED — WDK handles all signing for deposit
- Do NOT Promise.all approve+deposit — they must be sequential
- Do NOT use `ethers.utils.*` — ethers v6 uses flat namespace
