   - `using SafeERC20 for IERC20`

   - Events: `Deposited(bytes32 indexed escrowId, address depositor, uint256 amount)`, `Released(bytes32 indexed escrowId, address recipient, uint256 amount)`, `Refunded(bytes32 indexed escrowId, address depositor, uint256 amount)`, `Slashed(bytes32 indexed escrowId, address bountyPool, uint256 amount)`
   - `constructor(address _usdt) Ownable(msg.sender)` — sets `usdt = IERC20(_usdt)`

   - `release(bytes32 escrowId, address recipient) external onlyOwner nonReentrant` — requires PENDING, sets RELEASED, calls `safeTransfer` to recipient, emits Released
   - `refund(bytes32 escrowId) external onlyOwner nonReentrant` — requires PENDING, sets REFUNDED, calls `safeTransfer` to depositor, emits Refunded
   - `slash(bytes32 escrowId, address bountyPool) external onlyOwner nonReentrant` — requires PENDING, sets SLASHED, calls `safeTransfer` to bountyPool, emits Slashed

5. Create `contracts/scripts/deploy.ts`:
   ```typescript
   import { ethers } from "hardhat";
   import * as fs from "fs";
   import * as path from "path";

   async function main() {
     const usdtAddress = process.env.SEPOLIA_USDT_CONTRACT;
     if (!usdtAddress) throw new Error("SEPOLIA_USDT_CONTRACT env var required");

     console.log("Deploying FraudEscrow with USDT:", usdtAddress);

     const FraudEscrow = await ethers.getContractFactory("FraudEscrow");
     const contract = await FraudEscrow.deploy(usdtAddress);
     await contract.waitForDeployment();
     const address = await contract.getAddress();

     console.log("FraudEscrow deployed to:", address);

     const deployed = {
       sepolia: {
         FraudEscrow: address,
         usdt: usdtAddress,
         deployedAt: new Date().toISOString()
       }
     };

     const deployedPath = path.join(__dirname, "../deployed.json");
     fs.writeFileSync(deployedPath, JSON.stringify(deployed, null, 2));
     console.log("Written to:", deployedPath);
   }

   main().catch((error) => {
     console.error(error);
     process.exit(1);
   });
   ```
