  <action>
1. Create `agent/src/evidence.js` with these exports:
   - `writeCaseFile(listing, classificationResult, actionTaken)` — async, returns filepath string
   - `isCaseFileExists(url)` — async, returns boolean

2. Module setup (ESM, matching project conventions):
   ```javascript
   import { writeFile, mkdir, readdir } from 'node:fs/promises';
   import { join, dirname } from 'node:path';
   import { fileURLToPath } from 'url';
   import { createHash } from 'node:crypto';
   ```
   - `CASES_DIR` = `join(__dirname, '../cases')`
   - Reuse the same `urlHash(url)` function: SHA-256 of URL truncated to 16 hex chars

3. `isCaseFileExists(url)`:
   - Compute hash from url
   - Read directory listing of `CASES_DIR` via `readdir`
   - Return `true` if any filename contains the hash
   - Catch ENOENT (directory doesn't exist yet) and return `false`

4. `writeCaseFile(listing, classificationResult, actionTaken)`:
   - `await mkdir(CASES_DIR, { recursive: true })` at top (handles first run)
   - Filename: `{timestamp}-{platform}-{urlHash}.md`
     - timestamp: `new Date().toISOString().replace(/[:.]/g, '-')`
     - platform: `listing.platform.toLowerCase()`
     - urlHash: 16-char SHA-256 of listing.url
   - Generate markdown content with these sections:
     a. **Header**: `# Fraud Case: {platform} -- {category}`
     b. **Generated timestamp** and **Classification Source**
     c. **Listing Details table**: Platform, Seller, Price (USD), Face Value (USD), Price Delta (%), URL, Listing Date, Data Source, Screenshot (`_(not captured — scrapers collect structured data only)_`)
     d. **Red Flags** section: bulleted list from listing.redFlags, or "None detected"
     e. **Classification Result table**: Category (bold), Confidence (%), Classification Source
     f. **Reasoning** paragraph: classificationResult.reasoning
     g. **Enforcement Action table**: Action Taken, Threshold Met (YES/NO based on confidence >= parseInt(process.env.FRAUD_CONFIDENCE_THRESHOLD ?? '85')), Etherscan Link placeholder `_(pending escrow transaction)_`
     h. **Drafted Enforcement Text** section: call `draftEnforcementText(listing, classificationResult)`
     i. **Footer**: `Case file generated by Ducket AI Galactica` + Apache 2.0

5. `draftEnforcementText(listing, result)` — internal function:
   - SCALPING_VIOLATION: Takedown request text mentioning seller, price, priceDeltaPct, face value
   - LIKELY_SCAM: Platform report text mentioning below-face-value pricing
   - COUNTERFEIT_RISK: Public warning text mentioning red flags
   - LEGITIMATE: "No Action: Listing classified as legitimate. Monitoring continues."

6. Log to stderr: `[Evidence] Case file written: {filename}`
