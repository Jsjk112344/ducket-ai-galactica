    8. **dashboard/server/api.ts** — Express API server on port 3001 with three endpoints:
       - `GET /api/listings` — reads agent/memory/LISTINGS.md, extracts all fenced ```json blocks via regex `/```json\n([\s\S]*?)\n```/g`, flattens arrays, enriches each listing with classification data by looking up its case file in agent/cases/ using 16-char SHA-256 URL hash. Returns JSON array.
       - `GET /api/wallet` — reads contracts/deployed.json for deployer address and contract addresses. Queries Sepolia RPC (process.env.SEPOLIA_RPC_URL) for ETH balance and USDT balance using ethers.JsonRpcProvider + ethers.Contract with ABI `['function balanceOf(address) view returns (uint256)']`. Returns WalletInfo JSON. Wraps RPC calls in Promise.race with 5-second timeout; returns cached last-known values on timeout.
       - `GET /api/cases/:urlHash` — reads agent/cases/ directory, finds file containing the urlHash param, reads and returns its content. Returns 404 JSON if not found.
       - Use `import.meta.url` + `fileURLToPath` for path resolution (ESM-compatible, NOT __dirname).
       - Listen on port 3001.
       - Parse case file markdown to extract classification fields: category (from "Category |" line), confidence (from "Confidence |" line), reasoning (from "Reasoning |" line), actionTaken (from "Action Taken |" line), etherscanLink (first https://sepolia.etherscan.io URL found).
