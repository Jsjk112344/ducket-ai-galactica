  - Create page
  - Navigate to: `https://www.facebook.com/marketplace/search/?query=${encodeURIComponent(eventName)}`
  - Use `waitUntil: 'domcontentloaded'` (NOT networkidle â€” FB login modal fires on networkidle)
  - Timeout: `parseInt(process.env.FACEBOOK_TIMEOUT ?? '20000')`
  - **Dismiss login modal** (wrap each step in try/catch):
    ```javascript
    // Method 1: Escape key
    try { await page.keyboard.press('Escape'); await setTimeout(1000); } catch {}
    // Method 2: Click outside modal
    try { await page.mouse.click(10, 10); await setTimeout(1000); } catch {}
    // Method 3: Close button
    try { await page.click('[aria-label="Close"]', { timeout: 3000 }); } catch {}
    ```
  - Wait 2 seconds for DOM to settle: `await setTimeout(2000)`
  - **DOM extraction via page.evaluate:**
    ```javascript
    const rawListings = await page.evaluate(() => {
      // Multiple selector strategies â€” FB changes classnames frequently
      const selectors = [
        '[href*="/marketplace/item/"]',
        '[data-testid="marketplace_item"]',
        '[aria-label*="$"]',
      ];
      const tiles = new Set();
      for (const sel of selectors) {
        document.querySelectorAll(sel).forEach(el => tiles.add(el.closest('a') ?? el));
      }
      return Array.from(tiles).map(tile => ({
        text: tile.innerText ?? '',
        href: tile.href ?? tile.querySelector('a')?.href ?? '',
        price: tile.innerText?.match(/\$[\d,.]+/)?.[0] ?? '',
        seller: 'unknown',
      }));
    });
    ```

  - Log `[Facebook] Found ${result.length} listings from DOM`
  - Return array
- In catch: log `[Facebook] Error: ${err.message} â€” returning mock data`, return `getMockFacebook(eventName)`
- In finally: `await browser?.close()`
