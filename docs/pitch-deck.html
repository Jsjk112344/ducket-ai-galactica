How the Agent Decides

Multi-signal risk scoring — not just price. Five independent signals are weighted and combined. Ambiguous cases (composite 25-70) are escalated to Claude for deep analysis. Every signal is visible in the dashboard.

5-Signal Risk Engine

Each signal scored 0-100, weighted, combined into a composite risk score. Price alone does NOT determine fraud.

Pricing Risk (30%) Market-adjusted — demand tier shifts thresholds
Seller Trust (25%) Account age, transaction history, verification
Listing Quality (20%) Description depth, transfer method, section data
Temporal Pattern (15%) New account + high value, last-minute listings
Platform Trust (10%) StubHub > Viagogo > Facebook Marketplace
Ambiguous Zone → Claude API

Composite risk 25-70 triggers Claude with the full signal breakdown. Claude weighs signal interactions — e.g., 200% markup + sold-out event + verified seller = legitimate.

  • Claude sees all 5 signals, not just price
  • Structured JSON output — guaranteed valid schema
  • Claude fails → graceful fallback to rules-only result

Four enforcement outcomes

SCALPING VIOLATION

Excessive markup beyond demand-adjusted threshold. Escrow slashed to bounty pool.

LIKELY SCAM

Suspicious signals aggregate — bait pricing, new account, DM-only. Buyer gets full refund.

COUNTERFEIT RISK

Fair price but unverified seller — right price, wrong provenance. Buyer gets full refund.

LEGITIMATE

Signals align — fair market price, trusted seller, verified transfer. USDT released to seller.

Wallet Flow & Escrow

WDK handles all wallet operations — non-custodial, keys in memory only. The smart contract enforces four conditional outcomes driven entirely by the agent's classification. No human mediator. No platform fee.

Key point: The seller never touches buyer funds. The agent deposits via WDK, the agent dispatches settlement. Users see a clean dashboard — crypto complexity is invisible.

Architecture

Three-workspace monorepo. The agent runs a 5-minute cron loop: scrape three platforms, classify every listing, enforce escrow if confidence hits the gate. One command to run everything.
